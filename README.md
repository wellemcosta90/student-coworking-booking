# Student Coworking Booking System

A PHP and MySQL web application that allows students to book meeting and individual rooms in a coworking environment.

## Features

- User registration and login
- PHP sessions
- Role-based access: admin, organiser, attendee
- Room management
- Booking management
- Admin user management
- MySQL database integration
- Simple UI animation and hover effects

## Local Setup

1. Put the project folder inside XAMPP `htdocs` as `student-coworking-booking`.
2. Start Apache and MySQL in XAMPP.
3. Import `database/student_coworking.sql` into MySQL.
4. Open:

`http://localhost/student-coworking-booking/index.php`

## Database

Database name:

`student_coworking_db`

The clean SQL dump is:

`database/student_coworking.sql`
